# Subnet Calculator 🌐

A simple and interactive **Subnet Calculator** web application that helps users quickly calculate network information from an IP address and CIDR.

## 🚀 Features

* Calculate **Network Address**
* Calculate **Broadcast Address**
* Show **Subnet Mask**
* Display **Number of Hosts**
* Clean **Dark UI**
* Animated **Networking Background**

## 🛠 Built With

* HTML
* CSS
* JavaScript
